# Гра "Вгадай число"

## Опис
Програма загадує випадкове ціле число від 1 до 100, а гравець намагається його вгадати за обмежену кількість спроб - 7.
Спроби користувач вводить з клавіатури, а програма відповідає, чи число більше, менше або дорівнює загаданому.

## Як запустити
1. Склонуйте репозиторій:
   ```bash
   git clone https://github.com/brylinskyi/python.git

2. Перейдіть до директорії з проектом:
   ```bash
   cd python/
   ```
3. Запустіть гру:
   ```python main.py```
